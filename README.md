# Interactive Visual Planner

A premium, interactive calendar and task management component built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**. Designed to emulate a professional physical wall calendar while providing powerful digital tools for task scheduling and visual planning.

## 🚀 Key Features

### 📅 Advanced Calendar Core
- **Physical Wall Aesthetic**: Mirrors a high-end wall calendar with dynamic hero imagery that updates for every month.
- **Fluid Responsiveness**: Seamlessly adapts from ultra-wide monitors to mobile devices with a high-performance drawer system.
- **Timezone Aware**: Fully optimized for **IST (Indian Standard Time)** with clear indicators across the interface.

### ✅ Task & Agenda Management
- **Smart Task System**: Create, track, and manage tasks with ease. Includes completion toggles and visual progress indicators.
- **Localized Persistence**: Utilizes an optimized **Local Storage** strategy to ensure data remains safe across sessions with zero server latency.
- **Sync to Google Calendar**: Instantly export dated tasks to your Google Calendar with one click.

### 🎨 Premium UI/UX
- **Elegant Dark Mode**: Professional high-contrast dark theme with smooth, hardware-accelerated transitions.
- **Selection Intelligence**: Toggle between **Single Date** and **Range** selection modes for precise planning.
- **Toast Notifications**: Non-intrusive, real-time feedback for all user actions (creation, deletion, updates).
- **Glassmorphism Design**: Extensive use of backdrop-blur and refined typography for a modern, expansive feel.

## 🛠️ Tech Stack
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4.0
- **Animations**: Framer Motion
- **Date Logic**: date-fns
- **Icons**: Lucide React

## 🏁 Getting Started

### Prerequisites
- Node.js 18.x or higher
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Dhakshin2007/Dhakshin_TUF_Assignment
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
